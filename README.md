# iyankdesu.github.io

-----

Github pages ini dibuat dengan menggunakan Jekyll, Jekyll merupakan sebuah *static site generator* yang dibuat dengan bahasa pemprograman Ruby.

- [Ruby Official Site](https://www.ruby-lang.org/)
- [Jekyll Offical Site](https://jekyllrb.com/)

Github pages akan digunakan untuk menulis catatan dari berbagai *problem & solve* pada sistem operasi linux, khususnya Manjaro Linux dan juga tentang catatan pemprograman dan banyak lainnya.
