## Markdown

-----

Markdown adalah sebuah bahasa markup yang bisa digunakan untuk membuat halaman website, markdown memungkinkan untuk menulis konten HTML dengan mudah karena format penulisannya yang mudah untuk ditulis.


## Sejarah Markdown

-----

Markdown dibuat oleh John Gruber pada tahun 2004 dengan menggunakan bahasa pemprograman Perl, dengan tujuan agar orang - orang dapat menulis menggunakan format penulisan plain text yang mudah dibaca dan ditulis, yang mampu untuk diubah atau di konversi menjadi HTML atau XHTML. Markdown telah banyak digunakan oleh website - website besar seperti Github, dengan Github Pages halaman yang mulanya adalah file markdown akan di konversi ke file HTML sehingga dapat tampil pada halaman web browser.


## Menulis Markdown

-----

Untuk menulis markdown hanya diperlukan text editor biasa, namun lebih mudah jika menggunakan aplikasi yang telah dikhususkan untuk menulis dalam format markdown seperti Typora, Marktext, dan Retext.

Berikut adalah beberapa format penulisan markdown serta perbandingannya dengan HTML :

1. Paragraf dengan markdown

```markdown
Membuat paragraf dengan markdown tidak memerlukan tag tambahan <p> seperti pada HTML.
```

Pada HTML

```HTML
<p>Ini adalah paragraf dalam HTML</p>
```

-----

2. Heading dengan Markdown

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

Pada HTML

```HTML
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

-----

Dengan adanya markdown pembuatan halaman website menjadi sangat dimudahkan, terutama jika halaman website hanya mengandung informasi tanpa perlu banyak menggunakan styling pada teks - teksnya. Selain itu, markdown masih dapat digunakan untuk hal lainnya, seperti membuat catatan agar lebih interaktif, mempermudah penulisan naskah atau script, dan masih ada lainnya.

Salah satu contoh penggunaan markdown yang paling sering digunakan adalah website yang menuliskan artikel maupun blog seperti pada Github Pages, artikel dan blog dalam website ini pun dibuat dengan menggunakan markdown untuk mempermudah dalam penulisan konten didalamnya.
