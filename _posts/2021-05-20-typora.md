## Typora

-----

Typora adalah sebuah aplikasi markdown editor open-source yang dapat digunakan pada Windows, MacOS dan distribusi Linux. Typora memiliki design UI/UX yang menarik sehingga mudah untuk digunakan, dengan banyak fitur yang sangat memudahkan dalam penulisan markdown.

Jika umumnya ketika ingin membuat sebuah table harus membuatnya dengan manual, pada typora cukup tekan `ctrl + T` untuk membuat table, serta banyak hal lainnya yang dapat dengan mudah dibuat hanya dengan kombinasi tombol shortcut.

Typora memiliki sangat banyak tema yang dapat diakses di [https://theme.typora.io](https://theme.typora.io), cara instalasi temanya pun terbilang cukup mudah karena hanya memerlukan copy dan paste. Typora dapat di unduh di website [https://typora.io](https://typora.io) kemudian unduh sesuai sistem operasi yang digunakan.

Typora dibuat oleh Abner serta banyak kontributor open-source lainnya dari berbagai negara, informasi mengenai credits pada typora dapat diakses melalui aplikasinya pada menu `Help > Credits`. Dokumentasi yang disediakan oleh para pengembang typora pun terbilang sangat lengkap, sehingga untuk mulai belajar menggunakan typora tidaklah sulit, dokumentasi typora dapat di akses di [https://support.typora.io](https://support.typora.io).

Salah satu fitur terbaik typora adalah memungkinkan untuk mengeksport file *md* menjadi berbagai macam jenis file seperti *pdf*, *html*, *rtf*, *docx*, *epub* dan banyak lainnya hanya dengan sekali klik.



## Kelebihan dan Kekurangan

-----

### Kelebihan

1. Cross-platform, dapat digunakan diberbagai sistem operasi
2. UI/UX yang simpel dan modern
3. Mudah untuk digunakan
4. Gratis dan open-source
5. Dokumentasi lengkap

### Kekurangan

1. Tidak sepenuhnya open-source, masih ada komponennya yang tertutup
2. Ukuran file aplikasi lumayan besar

Secara keseluruhan typora adalah sebuah aplikasi markdown editor yang sangat bagus dan mudah digunakan, typora merupakan pilihan tepat bagi blogger yang menggunakan markdown untuk menampilkan halaman websitenya, para penulis naskah yang perlu fitur untuk eksport file plain text ke format *html*.
